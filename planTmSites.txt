## 4️⃣ JS Part For Learning & Quizzes

* **Accordion for terms** (as we did earlier)
* **Quiz interaction**:

```javascript
// pseudo JS for quiz
const quizForm = document.getElementById('quizForm');
quizForm.addEventListener('submit', e => {
    e.preventDefault();
    let score = 0;
    const answers = document.querySelectorAll('input[type="radio"]:checked');
    answers.forEach(answer => {
        if(answer.dataset.correct === "true") score++;
    });
    alert(`Your score: ${score} / ${answers.length}`);
});
```

* You can **expand this** to store results in the backend.

## 5️⃣ User Login & Backend Flow

1. **User registers/login** → form submits to backend → saves in DB
